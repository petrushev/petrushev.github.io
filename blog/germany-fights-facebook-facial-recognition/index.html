Germany starts a fight against Facebook facial recognition feature

published: 16 August, 2012

German officials accused Facebook for using user data and pictures for facial recognition without consent. Even though the user can opt-out, the European data protection law requires this kind of services to ask for explicit user permissions, that is, Facebook's defaults should be set to 'do not use my data'. Germany's demands are that Facebook destroy's all data collected from German citizens.
